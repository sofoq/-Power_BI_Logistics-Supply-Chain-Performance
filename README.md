# Logistics & Supply Chain Performance Analytics

## Project Overview
Designed and developed a comprehensive Business Intelligence (BI) solution to monitor and optimize supply chain operations. The primary objective of this project was to transform raw, fragmented operational data into actionable insights for logistics and warehouse management, enabling data-driven decision-making.

## Key Technical Highlights

* **Warehouse Management:** Engineered interactive dashboards to track storage capacity, real-time stock levels, and blocked inventory across multiple facilities, providing a holistic view of warehouse health.
* **Cost Analysis:** Implemented dynamic tracking of transportation expenses, including fixed pricing models, full-price structures, and diesel surcharges to monitor logistics profitability.
* **Data Engineering:** Built a robust data model using **Star Schema** architecture to ensure high-performance reporting, efficient data retrieval, and scalability.
* **Advanced Analytics:** Utilized **Advanced DAX** to develop complex time-intelligence metrics, performance KPIs, and support multi-layered ad-hoc reporting.
* **Visualization:** Integrated geospatial mapping and advanced conditional formatting to visualize shipment routes, logistics provider performance, and identify potential supply chain bottlenecks.

## Tech Stack
* **BI Tool:** Power BI Desktop
* **Data Transformation:** Power Query (M)
* **Modeling:** Star Schema / Relational Data Modeling
* **Language:** DAX (Data Analysis Expressions) & SQL
* **Visualization:** Interactive Maps, Heatmaps, and Custom KPI Cards
